This is a little app. built with :blibb and lungo.js.

The main use is to log all the bugs found during the development of :blibb, if you find an error, bug or want to suggest a feature, do not doubt to get in contact with us or just log the bug.

If you're interested on looking at the code you can always view the source or pop in to github and check our repo. There's a screencast as well with the "making of" of this app.
